# Multilingual Documentation

稷下观澜 aims to be a global project, so public documents should be readable beyond Chinese-speaking contributors.

## Canonical Source

The current canonical source language is **Simplified Chinese**:

- `README.md`
- `CHARTER.md`

English is the first global access language:

- `README.en.md`
- `CHARTER.en.md`

If translations conflict, the current Chinese `CHARTER.md` and `README.md` are authoritative until the community explicitly changes that rule.

The original text is Chinese. If any translation causes semantic misunderstanding, contributors are encouraged to maintain and correct it through Issues or PRs.

## Translation Principles

Translations must preserve these core commitments:

- This is a social experiment, not a traditional Skill store.
- The project is not commercialization-driven.
- The founder has no permanent privilege or reserved revenue right.
- Rules are public and can be changed through visible community process.
- Skill authorship and attribution must be preserved.
- A Skill should never permanently lose the right to be rediscovered or resurrected.

## Adding a Language

To add a language:

1. Create `README.{lang}.md`.
2. Create `CHARTER.{lang}.md`.
3. Add links to the top of `README.md`, `README.en.md`, and this file.
4. Keep governance meaning aligned with the Chinese source.
5. If the translation is incomplete, clearly mark it as a draft.

Use standard language tags where practical, such as `ja`, `ko`, `fr`, `es`, or `pt-BR`.
